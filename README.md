# Сонячні електростанції під ключ - Лендінг-сторінка

Сучасний, ефективний односторінковий лендінг для продажу сонячних електростанцій під ключ у Львові та Львівській області.

## Опис проекту

Лендінг-сторінка розроблена для компанії "АКУМУЛЯТОР-Центр", що спеціалізується на встановленні сонячних електростанцій потужністю 30-150 кВт для бізнесу. Сайт має стильний дизайн, оптимізований для мобільних пристроїв, та високу швидкість завантаження.

## Особливості

- **Адаптивний дизайн**: Повністю адаптивний під різні пристрої (мобільні, планшети, десктоп)
- **Оптимізація швидкості**: Швидке завантаження для покращення користувацького досвіду
- **Перемикач тем**: Три варіанти оформлення (світла, темна, синьо-жовта)
- **Калькулятор окупності**: Інтерактивний калькулятор для розрахунку економії та окупності
- **Форма зворотнього зв'язку**: Валідація полів, захист від спаму, відправка даних у Telegram
- **SEO-оптимізація**: Семантична структура, JSON-LD схеми для покращення індексації

## Структура сайту

1. **Hero секція**: Головний екран з заголовком і кнопками заклику до дії
2. **Болі та рішення**: Проблеми клієнтів та їх вирішення
3. **Послуги**: Перелік послуг компанії
4. **Процес роботи**: 5 кроків від замовлення до реалізації
5. **Обладнання**: Інформація про типи обладнання та гарантії
6. **Калькулятор окупності**: Розрахунок економії та терміну окупності
7. **Фінансування**: Інформація про кредити та гранти
8. **Кейси та результати**: Приклади успішних проектів
9. **Відгуки клієнтів**: Соціальні докази
10. **Часті запитання (FAQ)**: Відповіді на поширені запитання
11. **Команда**: Інформація про спеціалістів компанії
12. **Форма зворотнього зв'язку**: Заявка на розрахунок проекту
13. **Контакти та карта**: Інформація для зв'язку та локація компанії

## Технічні характеристики

- **HTML5, CSS3, JavaScript**: Сучасні веб-технології
- **Vanilla JS**: Без залежностей від зовнішніх бібліотек
- **Змінні CSS**: Легка зміна тем та адаптація дизайну
- **Телеграм інтеграція**: Відправка заявок напряму в Telegram
- **Аналітика**: Підготовлено для інтеграції з Google Analytics

## Інсталяція та запуск

1. Клонуйте репозиторій
2. Відкрийте файл `index.html` у браузері для локального перегляду
3. Для запуску на сервері, завантажте всі файли на ваш хостинг

## Налаштування

- **Контактні дані**: Відредагуйте контакти у файлі `index.html`
- **Telegram інтеграція**: Змініть BOT_TOKEN та CHAT_ID у файлі `js/main.js`
- **Google Analytics**: Додайте ваш ID відстеження у `index.html`

## Підготовка до продакшену

Перед розгортанням на робочому сервері:

1. Оптимізуйте зображення за допомогою інструментів типу TinyPNG
2. Мініфікуйте JS і CSS файли
3. Налаштуйте HTTPS
4. Оновіть мета-теги для SEO
